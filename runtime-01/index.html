§  RUNTIME · ISSUE NO. 01

The Code Mode Issue

A quarterly about machines that think — and the substrates that hold them

"The agent has always been writing."

01 Abstract

The industry has operated under a flawed assumption — that AI agents fundamentally work by calling tools from a menu. Cloudflare's 2025-26 releases demonstrate this model was misguided. The actual primitive is code itself.

02 From the Cover

RUNTIME is a quarterly about machines that think and the substrates that hold them. This issue is about code mode: what it is, why isolates matter more than containers, and how agent infrastructure becomes persistent rather than ephemeral.

Features:

  • The Code Mode Thesis — ~1,000-token footprint regardless of endpoint count
  • Inside the Dynamic Worker — an isolate, not a container
  • Project Think — an ephemeral agent is a tool; a durable agent is infrastructure
  • Memory as Infrastructure
